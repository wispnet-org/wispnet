About

Since 2024, the WISPNET has served as a structure for personal projects and open-source software.

History

The original WISPNET was a Local Area Network created as an experiment: two old OptiPlex servers connected to a small desktop network switch.
Now, it's a basement homebrew server rack, ready for whatever project is next.
